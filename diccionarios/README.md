# Diccionarios

Los diciionarios en python se delimitanpor corchetes "{ }", con los elementos separados por comas y la clave separada del valor mediante dos puntos. Veamos un ejemplo de diccionarios:

|Clave   |Valor   |
|--------|--------|
|"Clave1"|"Valor1"|
|"Clave2"|"Valor2"|
|"Clave3"|"Valor3"|
